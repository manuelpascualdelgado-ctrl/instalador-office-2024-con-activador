✨ Instalador y Configurador de Office 2024
Este paquete incluye dos scripts diseñados para instalar y configurar Office 2024 en equipos con licencias válidas.
El proceso es completamente automático y guiado mediante menús.

📦 Archivos incluidos


🚀 1. Instalación de Office 2024
🔧 Pasos:
1. 	Ejecuta instalar.bat.
2. 	Cuando Windows lo solicite, permite la ejecución como administrador.
3. 	El instalador se iniciará automáticamente.
4. 	Espera a que finalice el proceso.


🔐 2. Configuración posterior
🧭 Navegación del menú:
1. 	Ejecuta activación.bat como administrador.
2. 	En el primer menú, selecciona la opción 2.
3. 	En el segundo menú, selecciona la opción 1.
4. 	Espera a que el proceso termine.


✅ Finalización
Una vez completados ambos pasos, Office 2024 quedará instalado y configurado correctamente en tu equipo autorizado.
